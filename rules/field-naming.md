# 欄位名稱精確匹配規則

## 核心原則
欄位名稱必須**逐字元完全匹配**，包含單位後綴、括號、中英文。

## 單位後綴必須保留

常見後綴：`(元)`, `(%)`, `(億)`, `(張)`, `(千張)`, `(口)`

```xs
// 正確 ✓
GetField("外資買賣超(張)")
GetField("融資餘額(千張)")
GetField("漲跌幅(%)")
GetField("本益比")
GetField("每股淨值(元)")

// 錯誤 ✗ — 缺少單位
GetField("外資買賣超")      // 少了(張)
GetField("融資餘額")        // 少了(千張)
GetField("漲跌幅")          // 少了(%)
```

## q_ 前綴快捷欄位（GetQuote 專用）

| 快捷寫法 | 等同 |
|----------|------|
| `q_Close` | `GetQuote("成交價")` |
| `q_Open` | `GetQuote("開盤價")` |
| `q_High` | `GetQuote("最高價")` |
| `q_Low` | `GetQuote("最低價")` |
| `q_Volume` | `GetQuote("成交量")` |
| `q_DiffRate` | `GetQuote("漲跌幅(%)")` |
| `q_Ref` | `GetQuote("參考價")` |

## 欄位名稱驗證要點
1. 檢查是否有**單位後綴**（元/張/億/%）
2. 檢查**中文用字**是否完全正確
3. 檢查**括號**是否為半形 `()`
4. 不確定時，查閱 fields-reference 確認正確名稱
