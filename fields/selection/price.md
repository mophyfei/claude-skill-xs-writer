# Selection 價格欄位 (27)

> 用於選股腳本

| 欄位名稱 | English Code | 支援商品 | 備註 |
|----------|-------------|---------|------|
| Jensen | Jensen | 台股 美(ETF) | | Jensen指標用於衡量實際的投資回報超過基於其風險水平所預期的市場回報。 |
| SHARPE | SHARPE | 台股 美(ETF) | | Sharpe指標衡量投資相對於其風險所帶來的超額回報率，即投資回報超過無風險利率 |
| Treynor | Treynor | 台股 美(ETF) | | Treynor指標衡量投資相對於系統性風險的回報率。 |
| 上游股價指標 | UpStreamPriceIndex | 台股 | | 利用供應商的股價波動計算股價指數。 |
| 下游股價指標 | DownStreamPriceIndex | 台股 | | 利用客戶的股價波動計算股價指數。 |
| 參考價 | RefPrice | 台股 | | 當日交易的參考價。一般而言是前一交易日的收盤價，如果股票商品有除權息的話，則是除 |
| 同業股價指標 | CompetitorsPriceIndex | 台股 | | 利用同業競爭公司的股價波動計算股價指數。 |
| 均價 | AvgPrice | 台股 大盤 類股指數 香港股票 大陸股票 | | 每日的平均成交價。 |
| 投資建議目標價 | AnalystPriceTargets | 美(股票) | | 市場機構對個股的股價預估平均值。 |
| 振幅 | Range(％) | 台股 | | 當期股價的振幅。單位為%。 |
| 收盤價 | Close | 台股 香港股票 大陸股票 美(股票) | | K棒資料的收盤價(最後一個成交價位)。 |
| 最低價 | Low | 台股 香港股票 大陸股票 美(股票) | | K棒資料的最低成交價位。 |
| 最高價 | High | 台股 香港股票 大陸股票 美(股票) | | K棒資料的最高成交價位。 |
| 月平均收益率 | AvgReturnM | 台股 | | 最近21日的平均賺賠比例。 |
| 本益比 | PERatio | 台股 美(股票) | | 股票的本益比。 |
| 標準差 | StdDev | 台股 | | 使用一年的每日漲跌幅，計算標準差數值，用來衡量股票報酬率的波動性。數值越小表示波 |
| 波動率 | Volatility | 台股 選擇權 | | 選擇權／權證商品標的股的歷史波動率。 |
| 漲停價 | UpLimit | 台股 | | 當日的漲停價。 |
| 漲跌幅 | Change(％) | 台股 香港股票 大陸股票 美(股票) | | 當期股價的漲跌幅。單位是％。 |
| 真實範圍 | TrueRange | 台股 | | 股價的真實範圍(TrueRange)，數值是考慮到跳空所推算出來的股價真正震盪幅 |
| 真實範圍波幅 | TR％ | 台股 | | 真實範圍(TrueRange) / 昨收價 \* 100%。單位為%，假如真實範 |
| 股價淨值比 | PBRatio | 台股 美(股票) | | 計算公式：收盤價／每股淨值。 |
| 貝他值 | Beta | 台股 美(ETF) | | 個股的股價變動與市場大盤指數變動的相關性。 |
| 跌停價 | DownLimit | 台股 | | 當日的跌停價。 |
| 週平均收益率 | AvgReturnW | 台股 | | 最近5日的平均賺賠比例。 |
| 開盤價 | Open | 台股 香港股票 大陸股票 美(股票) | | K棒資料的開盤價(第一個成交價位)。 |
| 高低差 | Range | 台股 | | 當期的最高價 － 當期的最低價。 |

## 使用範例

```xs
close = GetField("收盤價");
high = GetField("最高價");
if close = high then ret = 1;
```

> 欄位名稱以 xs-helper 官方文件為準
