# Selection 財務-獲利欄位 (~70) - 多數僅台股

> 用於選股腳本 `GetField("欄位名")`
> **重要：欄位名稱包含單位後綴如 (%)、(元)、(億)，撰寫時務必完整填寫！**

## 獲利能力 (季/年)

| 欄位名稱 | 備註 |
|----------|------|
| 營業利益率(%) | 營業利益/營收 |
| 毛利率(%) | 毛利/營收 |
| 淨利率(%) | 稅後淨利/營收 |
| 稅前淨利率(%) | 稅前淨利/營收 |
| 營業利益(億) | 營業利益金額 |
| 稅後淨利(億) | 稅後淨利金額 |
| 稅前淨利(億) | 稅前淨利金額 |

## 報酬率

| 欄位名稱 | 備註 |
|----------|------|
| ROE(%) | 股東權益報酬率 |
| ROA(%) | 資產報酬率 |
| ROIC(%) | 投入資本報酬率 |

|----------|------|

| 每股盈餘累計(元) | EPS - 累計 |

| 每股營業利益(元) | 每股營業利益 |

## 成長率

| 欄位名稱 | 備註 |
|----------|------|
| 營利率年增率(%) | 營業利益率YoY |
| 毛利率年增率(%) | 毛利率YoY |

## 杜邦分析

| 欄位名稱 | 備註 |
|----------|------|
| 淨利率(%) | 稅後淨利/營收 |
| 總資產週轉率(次) | 營收/總資產 |
| 權益乘數(倍) | 總資產/股東權益 |

## 使用範例

```xs
// 高獲利成長股選股 - 注意欄位名稱含單位後綴
if GetField("ROE(%)") > 15
   AND GetField("毛利率(%)") > 40
   AND GetField("EPS年增率(%)") > 20
then ret = 1;
```

> **常見錯誤**：寫成 `GetField("ROE")` 而非 `GetField("ROE(%)")`，缺少單位後綴會導致找不到欄位。
> 完整欄位清單請查閱 xs-helper: ~/.cache/xs-helper/xs-helper backup/
