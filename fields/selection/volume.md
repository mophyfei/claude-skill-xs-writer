# Selection 量能欄位 (61)

> 用於選股腳本 `GetField("欄位名")`

## 基本量能

| 欄位名稱 | 支援商品 | 備註 |
|----------|---------|------|
| 成交量 | 全商品 | 當日成交量 |
| 成交金額 | 全商品 | 當日成交金額 |
| 內盤量 | 全商品 | 主動賣方量 |
| 外盤量 | 全商品 | 主動買方量 |
| 內外盤比 | 全商品 | 外盤/內盤 |
| 估計量 | 台股 | 預估總量 |
