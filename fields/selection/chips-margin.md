# Selection 籌碼-融資融券欄位 (19)

> 用於選股腳本
> 融資、融券、券資比、週轉率等相關欄位

| 欄位名稱 | English Code | 支援商品 | 備註 |
|----------|-------------|---------|------|
| 借券賣出還券張數 | SBStcokDiff | 台股 | | 償還先前借券賣出的張數。 |
| 券資比 | ShortSale／Margin | 台股 類股指數 | | 融券餘額與融資餘額的比例。計算公式：融券餘額張數 / 融資餘額張數 \* 100 |
| 融券使用率 | ShortSell％ | 台股 | | 融券使用的比例。計算公式：融券餘額 / 融資限額 \* 100%。 |
| 融券增減張數 | ShortSaleNew | 台股 類股指數 | | 融券餘額(張數)與前一期的異動張數。 |
| 融券買進張數 | ShortSaleRecall | 台股 | | 融券買進的張數。 |
| 融券賣出張數 | ShortSales | 台股 | | 融券賣出的張數。 |
| 融券餘額佔股本比例 | Shortsaleremain／Capital | 台股 | | 計算公式 : 融券餘額張數／發行張數\*100%。 |
| 融券餘額張數 | ShortSaleRemain | 台股 類股指數 | | 融券餘額張數，也就是到目前為止以融券賣出的淨張數。 |
| 融資使用率 | pomusingratio | 台股 | | 融資使用的比例。計算公式：融資餘額／融資限額 \* 100%。 |
| 融資增減張數 | pomnew | 台股 類股指數 | | 融資餘額(張數)與前一期的異動張數。 |
| 融資維持率 | Pomremainratio | 台股 | | 系統估算的大盤或是個股的融資維持率，也就是股票市值與融資金額的比例。 |
| 融資買進張數 | POMBuy | 台股 | | 融資買進的張數。 |
| 融資賣出張數 | POMSell | 台股 | | 融資買進的張數。 |
| 融資限額張數 | MarginLimit | 台股 | | 交易所公告的融資限額張數。 |
| 融資餘額佔股本比例 | Pomremain／Capital | 台股 | | 計算公式 : 融資餘額張數／發行張數\*100%。 |
| 融資餘額張數 | Pomremain | 台股 類股指數 | | 融資餘額張數，也就是到目前以融資買進的淨張數。 |
| 資券互抵張數 | MarginTrades | 台股 類股指數 | | 資券當沖(信用當沖)的張數。 |
| 週轉率 | TurnOverRatio | 台股 | | 股票交易量與流通股數的比例。 |
| 還券張數 | SBLreturn | 台股 | | 償還先前借券的張數。 |

## 使用範例

```xs
margin = GetField("融資增減張數");
if margin < -500 then ret = 1;  // 融資大減
```

> 欄位名稱以 xs-helper 官方文件為準
