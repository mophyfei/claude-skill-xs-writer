# Selection 財務-現金流欄位 (~43)

> 用於選股腳本 `GetField("欄位名")`
> **重要：欄位名稱包含單位後綴如 (%)、(元)、(億)，撰寫時務必完整填寫！**

## 營業現金流

| 欄位名稱 | 備註 |
|----------|------|
| 營業現金流量(億) | 營業活動現金流 |
| 每股營業現金流(元) | 每股營業現金流 |
| 營業現金流量年增率(%) | 營業CF YoY |
| 營業現金流量佔營收比(%) | 營業CF/營收 |
| 營業現金流量佔稅後淨利比(%) | CF品質比率 |

## 投資現金流

| 欄位名稱 | 備註 |
|----------|------|
| 投資現金流量(億) | 投資活動現金流 |
| 資本支出(億) | 資本支出金額 |
| 每股資本支出(元) | 每股資本支出 |
| 資本支出佔營收比(%) | CAPEX/營收 |

## 融資現金流

| 欄位名稱 | 備註 |
|----------|------|
| 融資現金流量(億) | 融資活動現金流 |
| 現金股利發放(億) | 現金股利總額 |

## 自由現金流

| 欄位名稱 | 備註 |
|----------|------|
| 自由現金流量(億) | 營業CF-資本支出 |
| 每股自由現金流(元) | 每股FCF |
| 自由現金流量營收比(%) | FCF/營收 |
| 自由現金流量年增率(%) | FCF YoY |
| 近四季自由現金流量(億) | 滾動四季合計 |

## 現金流品質

| 欄位名稱 | 備註 |
|----------|------|
| 現金流量允當比率(%) | CF充足率 |
| 現金再投資比率(%) | 再投資比率 |
| 淨現金流量(億) | 三大CF合計 |
| 期末現金(億) | 期末現金餘額 |

## 使用範例

```xs
// 現金流量充沛選股 - 注意欄位含單位後綴
if GetField("自由現金流量(億)") > 0
   AND GetField("營業現金流量佔營收比(%)") > 15
   AND GetField("每股自由現金流(元)") > 3
then ret = 1;
```

> **常見錯誤**：寫成 `GetField("自由現金流量")` 而非 `GetField("自由現金流量(億)")`。
> 完整欄位清單請查閱 xs-helper: ~/.cache/xs-helper/xs-helper backup/
