# Selection 財務-現金流欄位 (20)

> 用於選股腳本
> 現金流量表相關欄位

| 欄位名稱 | English Code | 支援商品 | 備註 |
|----------|-------------|---------|------|
| 來自營運之現金流量 | CashFlow－Operating | 台股 美(股票) | | 現金流量表上的「來自營運之現金流量」欄位。 |
| 因子_現金流量波動度 | Factor_VCF | 台股 | | 標準差(過去16季之營業現金流量/營業收入)。 |
| 因子分數_現金流量波動度 | FactorScore_VCF | 台股 | | 現金流量波動度因子經過過濾因子極端值、流動性不足及交易天數不足的商品後的優化排名 |
| 投資活動之現金流量 | CashFlowInvestment | 台股 美(股票) | | 現金流量表上的「投資活動之現金流量」欄位。 |
| 每股現金流量 | CashflowPerShare | 台股 | | 財務比率表上的「每股現金流量」欄位。 |
| 每股自由現金流量 | FCFFPerShare | 台股 | | 計算公式：自由現金流量／加權平均股數 |
| 淨值自由現金流量比 | NetValue／FCF | 台股 | | 計算公式: 股東權益總額 / 自由現金流量\*100%。 |
| 營業現金流量_營業利益 | CFO／OperatingIncome | 台股 | | 計算公式: 營業現金流量／營業利益\*100%。 |
| 營運資金 | WorkingCapital | 台股 | | 計算公式 : 流動資產 - 流動負債 |
| 現金再投資_ | CashReinvestmentRatio | 台股 | | 財務比率表上的"現金再投資%"欄位。 |
| 現金及約當現金 | CashCashEquivalent | 台股 | | 「資產負債表」中「流動資產」項內的「現金及約當現金」項目。 |
| 現金派息比率 | CurrentPayoutRatio | 台股 | | 計算公式: 現金股利 / 每股盈餘\*100%。 |
| 現金流量允當_ | CFAdequacyRatio | 台股 | | 計算公式 : 最近5年度來自營運之現金流量/ 最近5年度(長期投資(新增)+固定 |
| 現金流量比率 | CFO ／CL ％ | 台股 | | 財務比率表上的「現金流量比率」欄位。 |
| 理財活動之現金流量 | CashFlowFinancing | 台股 美(股票) | | 現金流量表上的「籌資活動之現金流量」欄位。 |
| 股價自由現金流量比 | Price／FCF | 台股 | | 計算公式: 股價 / 每股自由現金流量。 |
| 自由現金流量 | FCFF | 台股 | | 現金流量表中的「"來自營運之現金流量" + "購置不動產廠房設備（含預付）－CF |
| 自由現金流量營收比 | FCF／Sales | 台股 美(股票) | | 計算公式: 自由現金流量／營業收入淨額\*100%。 |
| 資本支出營收比 | Capitalexpenditure／Sales | 台股 | | 計算公式 : 現金流量表中 購置不動產廠房設備（含預付）/營業收入淨額\*100 |
| 資本支出金額 | CapitalExpenditure | 台股 | | 現金流量表內的"購置不動產廠房設備（含預付）－CFI"項目。現金流量表以負值表示 |

## 使用範例

```xs
fcf = GetField("自由現金流量");
if fcf > 0 then ret = 1;
```

> 欄位名稱以 xs-helper 官方文件為準
