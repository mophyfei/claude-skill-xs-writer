# Selection 財務-營收欄位 (~55)

> 用於選股腳本 `GetField("欄位名")`
> **重要：欄位名稱包含單位後綴如 (%)、(元)、(億)，撰寫時務必完整填寫！**

## 月營收

| 欄位名稱 | 備註 |
|----------|------|
| 月營收(億) | 最新月營收 |
| 月營收年增率(%) | 月營收YoY |
| 月營收月增率(%) | 月營收MoM |
| 累計營收年增率(%) | 累計營收YoY |
| 累計營收(億) | 本年度累計營收 |
| 月營收3個月平均年增率(%) | 3月均值YoY |
| 月營收12個月平均年增率(%) | 12月均值YoY |

## 季營收

| 欄位名稱 | 備註 |
|----------|------|
| 營業收入淨額(億) | 季營收淨額 |
| 營業收入年增率(%) | 季營收YoY |
| 營業收入季增率(%) | 季營收QoQ |
| 每股營業額(元) | 每股營收 |
| 每股營業額年增率(%) | 每股營收YoY |

## 營收結構

| 欄位名稱 | 備註 |
|----------|------|
| 營業成本(億) | 營業成本金額 |
| 營業費用(億) | 營業費用金額 |
| 營業外收支(億) | 營業外收入/支出 |
| 營業費用率(%) | 營業費用/營收 |
| 營業成本率(%) | 營業成本/營收 |

## 營收品質

| 欄位名稱 | 備註 |
|----------|------|
| 應收帳款週轉率(次) | 應收帳款周轉效率 |
| 存貨週轉率(次) | 存貨周轉效率 |
| 應收帳款天數(天) | 應收帳款回收天數 |
| 存貨天數(天) | 存貨周轉天數 |
| 營業週期(天) | 應收天數+存貨天數 |

## 使用範例

```xs
// 營收成長選股 - 注意欄位名稱含單位
if GetField("月營收年增率(%)") > 20
   AND GetField("累計營收年增率(%)") > 15
   AND GetField("月營收(億)") > GetField("月營收(億)")[1]
then ret = 1;
```

> **常見錯誤**：寫成 `GetField("月營收年增率")` 而非 `GetField("月營收年增率(%)")`。
> 完整欄位清單請查閱 xs-helper: ~/.cache/xs-helper/xs-helper backup/
