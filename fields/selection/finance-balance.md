# Selection 財務-資產負債欄位 (~45)

> 用於選股腳本 `GetField("欄位名")`
> **重要：欄位名稱包含單位後綴如 (%)、(元)、(億)、(倍)，撰寫時務必完整填寫！**

## 每股數據

| 欄位名稱 | 備註 |
|----------|------|
| 每股淨值(元) | BPS - 每股帳面價值 |
| 每股現金(元) | 每股現金及約當現金 |
| 每股有形淨值(元) | 扣除無形資產 |

## 資產

| 欄位名稱 | 備註 |
|----------|------|
| 總資產(億) | 資產總額 |
| 流動資產(億) | 流動資產合計 |
| 現金及約當現金(億) | 現金部位 |
| 短期投資(億) | 短期金融資產 |
| 應收帳款(億) | 應收帳款淨額 |
| 存貨(億) | 存貨金額 |
| 固定資產(億) | 不動產廠房設備 |
| 無形資產(億) | 無形資產金額 |
| 長期投資(億) | 長期投資金額 |

## 負債

| 欄位名稱 | 備註 |
|----------|------|
| 總負債(億) | 負債總額 |
| 流動負債(億) | 流動負債合計 |
| 長期負債(億) | 長期借款 |
| 短期借款(億) | 短期借款金額 |
| 應付帳款(億) | 應付帳款金額 |

## 股東權益

| 欄位名稱 | 備註 |
|----------|------|
| 股東權益(億) | 股東權益合計 |
| 保留盈餘(億) | 保留盈餘合計 |

## 財務比率

| 欄位名稱 | 備註 |
|----------|------|
| 負債比率(%) | 總負債/總資產 |
| 流動比率(%) | 流動資產/流動負債 |
| 速動比率(%) | (流動資產-存貨)/流動負債 |
| 利息保障倍數(倍) | EBIT/利息支出 |
| 長期資金適合率(%) | 長期資金/固定資產 |
| 淨負債比率(%) | 淨負債/股東權益 |
| 有息負債比率(%) | 有息負債/總資產 |

## 資產品質

| 欄位名稱 | 備註 |
|----------|------|
| 總資產週轉率(次) | 營收/總資產 |
| 固定資產週轉率(次) | 營收/固定資產 |
| 淨值週轉率(次) | 營收/股東權益 |

## 使用範例

```xs
// 財務穩健選股 - 注意欄位含單位後綴
if GetField("負債比率(%)") < 50
   AND GetField("流動比率(%)") > 150
   AND GetField("利息保障倍數(倍)") > 5
then ret = 1;
```

> **常見錯誤**：寫成 `GetField("負債比率")` 而非 `GetField("負債比率(%)")`。
> 完整欄位清單請查閱 xs-helper: ~/.cache/xs-helper/xs-helper backup/
