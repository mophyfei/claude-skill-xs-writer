# Selection 常用欄位 (19)

> 用於選股腳本 `GetField("欄位名")`

| 欄位名稱 | English Code | 支援商品 | 備註 |
|----------|-------------|---------|------|
| 主力買賣超張數 | LeaderDifference | 台股 | | 主力買張 ─ 主力賣張。 |
| 均價 | AvgPrice | 台股 大盤 類股指數 香港股票 大陸股票 | | 每日的平均成交價。 |
| 成交量 | Volume | 台股 香港股票 大陸股票 美(股票) | | K棒資料的成交量。 |
| 成交金額(億) | TradeValue | 台股 類股指數 | | 成交的加總金額，換算成億。 |
| 收盤價 | Close | 台股 香港股票 大陸股票 美(股票) | | K棒資料的收盤價(最後一個成交價位)。 |
| 最低價 | Low | 台股 香港股票 大陸股票 美(股票) | | K棒資料的最低成交價位。 |
| 最高價 | High | 台股 香港股票 大陸股票 美(股票) | | K棒資料的最高成交價位。 |
| 月營收 | Sales | 台股 | | 台股上市櫃公司每月公告的營業收入數值。 |
| 本益比 | PERatio | 台股 美(股票) | | 股票的本益比。 |
| 殖利率 | Yield | 台股 美(股票) 美(特別股) 美(ETF) | | 股票的殖利率。單位是%，例如殖利率為5.6％的話，則回傳5.6。 |
| 每股稅後淨利(元) | EPS | 台股 美(股票) | | 台股，是損益表上的「每股盈餘」項目。 |
| 法人買賣超張數 | InvestorDifference | 台股 類股指數 | | 三大法人(外資、投信、自營商)合計買賣超張數。 |
| 漲跌幅 | Change(％) | 台股 香港股票 大陸股票 美(股票) | | 當期股價的漲跌幅。單位是％。 |
| 營業毛利率 | GrossMargin | 台股 美(股票) | | 財務比率表上的「營業毛利率」欄位。 |
| 總市值(億) | MarketCapin100Million | 台股 類股指數 | | 股票或是指數的市值，換算成億元。股票市值以發行股數 ＊ 收盤價來計算，指數則是成 |
| 股價淨值比 | PBRatio | 台股 美(股票) | | 計算公式：收盤價／每股淨值。 |
| 股東權益報酬率 | ROE | 台股 美(股票) | | 台股，是財務比率表的「ROE(A)─稅後」項目。 |
| 轉換價格 | ConversionPrice | 台(可轉債) | |  |
| 開盤價 | Open | 台股 香港股票 大陸股票 美(股票) | | K棒資料的開盤價(第一個成交價位)。 |

## 使用範例

```xs
value1 = GetField("收盤價");  // 或 GetField("Close")
value2 = GetField("成交量");  // 或 GetField("Volume")
if value1 > 50 and value2 > 1000 then ret = 1;
```

> 欄位名稱以 xs-helper 官方文件為準
