# Selection 常用欄位 (19)

> 用於選股腳本 `GetField("欄位名")` 或直接比較

| 欄位名稱 | 支援商品 | 備註 |
|----------|---------|------|
| 收盤價 | 全商品 | 最近一日收盤價 |
| 開盤價 | 全商品 | 最近一日開盤價 |
| 最高價 | 全商品 | 最近一日最高價 |
| 最低價 | 全商品 | 最近一日最低價 |
| 成交量 | 全商品 | 最近一日成交量 |
| 成交金額 | 全商品 | 最近一日成交金額 |
| 漲幅(%) | 全商品 | 漲跌幅百分比 |
| 振幅(%) | 全商品 | 當日振幅 |
| 外資買賣超 | 台股 | 外資淨買賣張數 |
| 投信買賣超 | 台股 | 投信淨買賣張數 |
| 自營商買賣超 | 台股 | 自營商淨買賣 |
| 三大法人買賣超 | 台股 | 合計淨買賣 |
| 融資增減 | 台股 | 融資增減張數 |
| 融券增減 | 台股 | 融券增減張數 |
| 主力買賣超 | 台股 | 主力淨買賣張數 |
| 週轉率 | 台股 | 成交量/流通股(%) |
| 均價 | 全商品 | 當日均價 |
| 漲跌 | 全商品 | 漲跌金額 |
| 量比 | 全商品 | 當日量/前日量 |

## 使用範例

```xs
// 基本選股條件
if GetField("收盤價") > 50
   AND GetField("成交量") > 1000
   AND GetField("外資買賣超") > 0
then ret = 1;
```

> 完整欄位清單請查閱 xs-helper: ~/.cache/xs-helper/xs-helper backup/
