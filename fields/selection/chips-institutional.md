# Selection 籌碼-法人欄位 (~40) - 僅台股

> 用於選股腳本 `GetField("欄位名")`，僅支援台股

## 外資

| 欄位名稱 | 備註 |
|----------|------|
| 外資買賣超 | 外資淨買賣張數 |
| 外資買張 | 外資買進張數 |
| 外資賣張 | 外資賣出張數 |
| 外資持股張數 | 外資持有張數 |
| 外資持股比率 | 外資持股占比(%) |
| 外資買賣超(累計) | N日累計 |
| 外資持股比率增減 | 持股比增減(%) |
| 外資買賣超金額 | 淨買賣金額 |

## 投信

| 欄位名稱 | 備註 |
|----------|------|
| 投信買賣超 | 投信淨買賣張數 |
| 投信買張 | 投信買進張數 |
| 投信賣張 | 投信賣出張數 |
| 投信持股張數 | 投信持有張數 |
| 投信持股比率 | 投信持股占比(%) |
| 投信買賣超(累計) | N日累計 |
| 投信買賣超金額 | 淨買賣金額 |

## 使用範例

```xs
// 外資投信同步買超選股
if GetField("外資買賣超") > 500
   AND GetField("投信買賣超") > 100
then ret = 1;
```

> 完整欄位清單請查閱 xs-helper: ~/.cache/xs-helper/xs-helper backup/
