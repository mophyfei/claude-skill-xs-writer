# Data 價格欄位 (14)

> 用於 `GetField("欄位名")` 歷史資料

| 欄位名稱 | English Code | 支援商品 | 備註 |
|----------|-------------|---------|------|
| 內外盤 | BidAskFlag | 全部 | | 這一筆成交明細內的「內外盤」標記。 |
| 參考價 | RefPrice | 台股 | | 當日交易的參考價。一般而言是前一交易日的收盤價，如果股票商品有除權息的話，則是除 |
| 均價 | AvgPrice | 台股 大盤 類股指數 期貨 選擇權 香港股票 大陸股票 | | 每日的平均成交價。 |
| 基差 | Basis | 大盤 期貨 | | 現貨價格與期貨價格之間的差異。 |
| 強弱指標 | StrongWeak | 台股 類股指數 | | 商品漲跌幅與對應大盤漲跌幅的差異，用來衡量商品是否比大盤強。 |
| 投資建議目標價 | AnalystPriceTargets | 美(股票) | | 市場機構對個股的股價預估平均值。 |
| 收盤價 | Close | 全部 | | K棒資料的收盤價(最後一個成交價位)。 |
| 最低價 | Low | 全部 | | K棒資料的最低成交價位。 |
| 最高價 | High | 全部 | | K棒資料的最高成交價位。 |
| 漲停價 | UpLimit | 台股 | | 當日的漲停價。 |
| 買入價 | BidPrice | 全部 | | 這一筆成交明細內的「買進」價格。 |
| 賣出價 | AskPrice | 全部 | | 這一筆成交明細內的「賣出」價格。 |
| 跌停價 | DownLimit | 台股 | | 當日的跌停價。 |
| 開盤價 | Open | 全部 | | K棒資料的開盤價(第一個成交價位)。 |

## 使用範例

```xs
avg = GetField("均價");  // 或 GetField("AvgPrice")
ref = GetField("參考價");
```

> 欄位名稱以 xs-helper 官方文件為準
