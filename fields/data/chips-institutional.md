# Data 籌碼-法人欄位 (~50) - 僅台股

> 用於 `GetField("欄位名")` 歷史資料，僅支援台股

## 外資

| 欄位名稱 | 備註 |
|----------|------|
| 外資買賣超 | 外資淨買賣張數(含外資自營) |
| 外資買張 | 外資買進張數 |
| 外資賣張 | 外資賣出張數 |
| 外資買賣超(累計) | N日累計外資買賣超 |
| 外資持股張數 | 外資持有張數 |
| 外資持股比率 | 外資持股占比(%) |
| 外資持股市值 | 外資持股市值 |
| 外資買賣超金額 | 外資淨買賣金額 |

## 投信

| 欄位名稱 | 備註 |
|----------|------|
| 投信買賣超 | 投信淨買賣張數 |
| 投信買張 | 投信買進張數 |
| 投信賣張 | 投信賣出張數 |
| 投信買賣超(累計) | N日累計投信買賣超 |
| 投信持股張數 | 投信持有張數 |
| 投信持股比率 | 投信持股占比(%) |
| 投信買賣超金額 | 投信淨買賣金額 |

## 自營商

| 欄位名稱 | 備註 |
|----------|------|
| 自營商買賣超 | 自營商淨買賣張數 |
| 自營商買張 | 自營商買進張數 |
| 自營商賣張 | 自營商賣出張數 |
| 自營商買賣超(累計) | N日累計 |
| 自營商(自行)買賣超 | 自營自行交易 |
| 自營商(避險)買賣超 | 自營避險交易 |

## 三大法人合計

| 欄位名稱 | 備註 |
|----------|------|
| 三大法人買賣超 | 三大法人合計淨買賣 |
| 三大法人買張 | 合計買進張數 |
| 三大法人賣張 | 合計賣出張數 |
| 三大法人買賣超(累計) | N日累計 |

## 使用範例

```xs
// 法人同步買超
foreign = GetField("外資買賣超");
trust = GetField("投信買賣超");
dealer = GetField("自營商買賣超");
if foreign > 0 AND trust > 0 AND dealer > 0 then
  ret = 1;  // 三大法人同步買超
```

> 完整欄位清單請查閱 xs-helper: ~/.cache/xs-helper/xs-helper backup/
