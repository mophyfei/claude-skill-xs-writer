# Data 籌碼-主力欄位 (~35) - 僅台股

> 用於 `GetField("欄位名")` 歷史資料，僅支援台股

## 主力買賣超

| 欄位名稱 | 備註 |
|----------|------|
| 主力買賣超 | 主力淨買賣張數 |
| 主力買張 | 主力買進張數 |
| 主力賣張 | 主力賣出張數 |
| 主力買賣超(累計) | N日累計主力買賣超 |
| 主力買進家數 | 主力買進券商家數 |
| 主力賣出家數 | 主力賣出券商家數 |
| 主力買賣家數差 | 買進-賣出家數 |

## 分點主力

| 欄位名稱 | 備註 |
|----------|------|
| 主力進出(1) | 第1大主力淨買賣 |
| 主力進出(2) | 第2大主力淨買賣 |
| 主力進出(3) | 第3大主力淨買賣 |
| 主力進出(4) | 第4大主力淨買賣 |
| 主力進出(5) | 第5大主力淨買賣 |
| 前5大主力合計 | 前5大主力淨買賣合計 |
| 前10大主力合計 | 前10大主力淨買賣合計 |
| 前15大主力合計 | 前15大主力淨買賣合計 |

## 主力集中度

| 欄位名稱 | 備註 |
|----------|------|
| 主力集中度 | 主力買超集中比率(%) |
| 買超集中度 | 買超券商集中度 |
| 賣超集中度 | 賣超券商集中度 |
| 主力成本 | 主力平均買進成本 |

## 使用範例

```xs
// 主力連續買超且集中度高
majorNet = GetField("主力買賣超");
majorConc = GetField("主力集中度");
if majorNet > 500 AND majorNet[1] > 500 AND majorConc > 50 then
  ret = 1;
```

> 完整欄位清單請查閱 xs-helper: ~/.cache/xs-helper/xs-helper backup/
