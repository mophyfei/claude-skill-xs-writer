## 使用範例

```xs
// 集保大戶增加 + 董監持股穩定
bigHolder = GetField("千張以上持股比率");
boardHold = GetField("董監持股比率");
if bigHolder > bigHolder[1] AND boardHold > 10 then
  ret = 1;
```
