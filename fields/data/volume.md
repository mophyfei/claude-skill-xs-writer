# Data 量能欄位 (69)

> 用於 `GetField("欄位名")` 歷史資料
> 注意：大部分欄位僅支援台股

## 基本量能

| 欄位名稱 | 支援商品 | 備註 |
|----------|---------|------|
| 成交量 / Volume | 全商品 | 當根K棒成交量 |
| 成交金額 / Amount | 全商品 | 當根K棒成交金額 |
| 內盤量 | 全商品 | 主動賣方成交量 |
| 外盤量 | 全商品 | 主動買方成交量 |
| 估計量 | 台股 | 預估總量 |
| 內外盤比 | 全商品 | 外盤量/內盤量 |

## 漲跌相關

| 欄位名稱 | 支援商品 | 備註 |
|----------|---------|------|
| 上漲量 | 台股 | 上漲時的成交量 |
| 下跌量 | 台股 | 下跌時的成交量 |
| 平盤量 | 台股 | 平盤時的成交量 |

| 均張 | 台股 | 平均每筆成交張數 |

| 5日均量 | 全商品 | 5日平均成交量 |

| 20日均量 | 全商品 | 20日平均成交量 |
| 60日均量 | 全商品 | 60日平均成交量 |

## 量比相關

| 欄位名稱 | 支援商品 | 備註 |
|----------|---------|------|
| 量比 | 全商品 | 當日量/前日量 |
| 量增(%) | 全商品 | 量增減百分比 |
| 5日量比 | 全商品 | 當日量/5日均量 |
| 10日量比 | 全商品 | 當日量/10日均量 |
| 20日量比 | 全商品 | 當日量/20日均量 |
