# Data 籌碼-融資融券欄位 (~40) - 僅台股

> 用於 `GetField("欄位名")` 歷史資料，僅支援台股

## 融資

| 欄位名稱 | 備註 |
|----------|------|
| 融資張數 | 融資餘額(張) |
| 融資餘額 | 融資餘額(金額) |
| 融資增減 | 融資增減張數 |
| 融資買進 | 融資買進張數 |
| 融資賣出 | 融資賣出張數 |
| 融資現償 | 融資現金償還 |
| 融資使用率 | 融資使用率(%) |
| 融資限額 | 融資額度上限 |
| 融資增減(累計) | N日累計融資增減 |

## 融券

| 欄位名稱 | 備註 |
|----------|------|
| 融券張數 | 融券餘額(張) |
| 融券餘額 | 融券餘額(金額) |
| 融券增減 | 融券增減張數 |
| 融券賣出 | 融券賣出張數 |
| 融券買進 | 融券回補張數 |
| 融券現償 | 融券現金償還 |
| 融券使用率 | 融券使用率(%) |
| 融券限額 | 融券額度上限 |
| 融券增減(累計) | N日累計融券增減 |

## 券資比與其他

| 欄位名稱 | 備註 |
|----------|------|
| 券資比 | 融券/融資(%) |
| 資券互抵 | 資券互抵張數 |
| 資券互抵率 | 資券互抵比率(%) |
| 融資維持率 | 融資維持率(%) |
| 當日沖銷率 | 當沖比率(%) |
| 借券賣出 | 借券賣出張數 |
| 借券餘額 | 借券餘額張數 |

## 使用範例

```xs
// 融資減 + 融券增 = 潛在軋空
marginBuy = GetField("融資增減");
shortSell = GetField("融券增減");
if marginBuy < 0 AND shortSell > 100 then
  ret = 1;
```

> 完整欄位清單請查閱 xs-helper: ~/.cache/xs-helper/xs-helper backup/
