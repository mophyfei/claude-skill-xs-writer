# Data 籌碼-融資融券欄位 (21)

> 用於 `GetField("欄位名")` 歷史資料
> 融資、融券、券資比等相關欄位

| 欄位名稱 | English Code | 支援商品 | 備註 |
|----------|-------------|---------|------|
| 借券賣出還券張數 | SBStcokDiff | 台股 大盤 | | 償還先前借券賣出的張數。 |
| 券資比 | ShortSale／Margin | 台股 大盤 類股指數 | | 融券餘額與融資餘額的比例。計算公式：融券餘額張數 / 融資餘額張數 \* 100 |
| 融券使用率 | ShortSell％ | 台股 | | 融券使用的比例。計算公式：融券餘額 / 融資限額 \* 100%。 |
| 融券增減張數 | Shortsalenew | 台股 大盤 類股指數 大陸股票 | | 融券餘額(張數)與前一期的異動張數。 |
| 融券買進張數 | ShortSaleRecall | 台股 大盤 | | 融券買進的張數。 |
| 融券賣出張數 | ShortSales | 台股 大盤 | | 融券賣出的張數。 |
| 融券餘額張數 | Shortsaleremain | 台股 大盤 類股指數 大陸股票 | | 融券餘額張數，也就是到目前為止以融券賣出的淨張數。 |
| 融資使用率 | Pomusingratio | 台股 | | 融資使用的比例。計算公式：融資餘額／融資限額 \* 100%。 |
| 融資增減 | pomDiff | 台股 大盤 類股指數 | | 融資餘額與前一期的異動。數值內容/單位依商品類型而異。 |
| 融資增減張數 | Pomnew | 台股 大盤 類股指數 大陸股票 | | 融資餘額(張數)與前一期的異動張數。 |
| 融資增減金額 | IndexPOMDiff | 大盤 | | 融資餘額(金額)與前一期的異動金額。 |
| 融資維持率 | Pomremainratio | 台股 大盤 | | 系統估算的大盤或是個股的融資維持率，也就是股票市值與融資金額的比例。 |
| 融資買進張數 | POMBuy | 台股 大盤 | | 融資買進的張數。 |
| 融資買進金額 | IndexPOMBuySideAmount | 大盤 | | 融資買進的總金額。 |
| 融資賣出張數 | POMSell | 台股 大盤 | | 融資買進的張數。 |
| 融資賣出金額 | IndexPOMSellSideAmount | 大盤 | | 融資賣出的總金額。 |
| 融資餘額 | Pomremainbycomm | 台股 大盤 類股指數 | | 融資餘額數值。數值內容/單位依商品類型而異。 |
| 融資餘額張數 | Pomremain | 台股 大盤 類股指數 大陸股票 | | 融資餘額張數，也就是到目前以融資買進的淨張數。 |
| 融資餘額金額 | IndexPOMRemain | 大盤 | | 融資餘額金額，也就是到目前以融資買進的淨金額。 |
| 資券互抵張數 | MarginTrades | 台股 大盤 類股指數 | | 資券當沖(信用當沖)的張數。 |
| 還券張數 | SBLreturn | 台股 大盤 | | 償還先前借券的張數。 |

## 使用範例

```xs
margin = GetField("融資增減張數");  // 或 GetField("Pomnew")
short_ = GetField("融券增減張數");  // 或 GetField("Shortsalenew")
```

> 欄位名稱以 xs-helper 官方文件為準
