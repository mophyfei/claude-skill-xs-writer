# Data 基本欄位 (9)

> 用於 `GetField("欄位名")` 歷史資料

| 欄位名稱 | 支援商品 | 備註 |
|----------|---------|------|
| 股本 | 台股/港股/陸股/美股 | 公司股本(元) |
| 市值 | 台股/港股/陸股/美股 | 公司市值 |
| 上市日期 | 台股 | 上市櫃掛牌日 |
| 產業別 | 台股/港股/陸股/美股 | 所屬產業分類 |
| 股票名稱 | 全商品 | 商品中文名稱 |
| 股票代號 | 全商品 | 商品代號 |
| 市場別 | 台股 | 上市/上櫃/興櫃 |
| 在外流通股數 | 台股/美股 | 流通在外股數 |
| 面額 | 台股 | 股票面額 |

## 使用範例

```xs
// 取得市值篩選
cap = GetField("市值");
if cap > 10000000000 then
  ret = 1;  // 市值大於100億
```

> 完整欄位清單請查閱 xs-helper: ~/.cache/xs-helper/xs-helper backup/
