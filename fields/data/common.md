# Data 常用欄位 (17)

> 用於 `GetField("欄位名")` 歷史資料

| 欄位名稱 | English Code | 支援商品 | 備註 |
|----------|-------------|---------|------|
| 估計量 | EstimateVolume | 大盤 類股指數 台(股票) | | 當日收盤的預估成交量。 |
| 內盤量 | TradeVolumeAtBid | 台股 大盤 類股指數 期貨 選擇權 香港股票 大陸股票 | | 這一筆K棒的所有成交內標記為內盤成交的加總數量。 |
| 參考價 | RefPrice | 台股 | | 當日交易的參考價。一般而言是前一交易日的收盤價，如果股票商品有除權息的話，則是除 |
| 均價 | AvgPrice | 台股 大盤 類股指數 期貨 選擇權 香港股票 大陸股票 | | 每日的平均成交價。 |
| 外盤量 | TradeVolumeAtAsk | 台股 大盤 類股指數 期貨 選擇權 香港股票 大陸股票 | | 這一筆K棒的所有成交內被標記為外盤成交的加總數量。 |
| 成交量 | Volume | 全部 | | K棒資料的成交量。 |
| 成交金額(元) | TradeValue | 台股 大盤 類股指數 香港股票 大陸股票 美(股票) | | 成交的加總金額。 |
| 收盤價 | Close | 全部 | | K棒資料的收盤價(最後一個成交價位)。 |
| 日期 | Date | 全部 | | K棒資料的日期。欄位格式為西元年月日，例如20221101。 |
| 時間 | Time | 全部 | | K棒資料的時間(時分秒)。格式是hhmmss的6位數字，例如103000代表10 |
| 最低價 | Low | 全部 | | K棒資料的最低成交價位。 |
| 最高價 | High | 全部 | | K棒資料的最高成交價位。 |
| 漲停價 | UpLimit | 台股 | | 當日的漲停價。 |
| 買入價 | BidPrice | 全部 | | 這一筆成交明細內的「買進」價格。 |
| 賣出價 | AskPrice | 全部 | | 這一筆成交明細內的「賣出」價格。 |
| 跌停價 | DownLimit | 台股 | | 當日的跌停價。 |
| 開盤價 | Open | 全部 | | K棒資料的開盤價(第一個成交價位)。 |

## 使用範例

```xs
value1 = GetField("收盤價");  // 或 GetField("Close")
value2 = GetField("成交量");  // 或 GetField("Volume")
```

> 欄位名稱以 xs-helper 官方文件為準
