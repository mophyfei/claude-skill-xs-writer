# Quote 量能欄位 (27)

> 用於 `GetQuote("欄位名")` 即時報價

| 欄位名稱 | English Code | 支援商品 | 備註 |
|----------|-------------|---------|------|
| 單量 | — | 全商品 | 最新一筆成交量 |
| 內盤量 | — | 全商品 | 累計內盤成交量 |
| 外盤量 | — | 全商品 | 累計外盤成交量 |
| 買賣力道 | — | 台股 | 買賣力道差值 |
| 內外盤比 | — | 全商品 | 外盤量/內盤量 |
| 總量(日) | TotalVolume | 全商品 | 當日累計成交量 |
| 昨量 | YVolume | 全商品 | 昨日總成交量 |
| 估計量 | EstimatedTotalVolume | 台股/大盤 | 預估總成交量 |
| 總額 | — | 全商品 | 當日累計成交金額 |
| 量比 | — | 全商品 | 當日量/昨日量 |
| 量增(%) | — | 全商品 | 量增減百分比 |
| 委買量 | — | 全商品 | 委買總量 |
| 委賣量 | — | 全商品 | 委賣總量 |
| 單筆成交量 | — | 全商品 | 同「單量」 |
| 總量(夜) | NightTotalVolume | 期貨 | 夜盤累計成交量 |
| 內盤量(日) | — | 全商品 | 日盤內盤量 |
| 外盤量(日) | — | 全商品 | 日盤外盤量 |
| 內盤量(夜) | — | 期貨 | 夜盤內盤量 |
| 外盤量(夜) | — | 期貨 | 夜盤外盤量 |
| 週轉率 | — | 台股 | 成交量/股本 |
| 均張 | — | 台股 | 平均每筆成交張數 |
| 成交筆數 | — | 台股 | 當日成交筆數 |
| 買進筆數 | — | 台股 | 委買筆數 |
| 賣出筆數 | — | 台股 | 委賣筆數 |
| 總委買筆 | — | 台股 | 委買總筆數 |
| 總委賣筆 | — | 台股 | 委賣總筆數 |
| 未平倉量 | — | 期貨/選擇權 | 未平倉合約數 |

## 使用範例

```xs
vol = GetQuote("TotalVolume");
yVol = GetQuote("YVolume");
if vol > yVol * 1.5 then ret = 1;
```

> 完整欄位清單請查閱 xs-helper: ~/.cache/xs-helper/xs-helper backup/
